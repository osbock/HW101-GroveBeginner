## Pin Numbers
 * led = 4
 * buzzer = 5;
 * button = 6;
 * rotary = A0;
 * light = A6;
 * sound = A2;
 * digital temp/humidity (DHT) = 3

LIS3DHTR accelerometer, OLED display, air pressure sensor all use I2C
